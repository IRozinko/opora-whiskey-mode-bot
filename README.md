# Опора / Whiskey Mode Telegram Bot

Telegram-бот MVP для системы «Опора / Дорогой виски»: рутины, голос, коммуникация, стиль, финансы, кредитки, резерв и авто-фонд.

## Что уже есть в MVP

### Рутины и коммуникация

- `/start` — запуск и список команд.
- `/morning` — утренний ритуал.
- `/evening` — вечерний разбор.
- `/status` — статус вечера для семьи.
- `/phrase` — AI-помощь с формулировкой фраз.
- `/silent` — разбор момента молчания.
- `/voice` — упражнение на голос и дикцию.
- `/look` — базовые правила образа.
- `/reset` — быстрый антиперегруз.
- `/podcast` — аудио-рекомендация.

### Финансы

- `/expense 245 happy meal сыну` — записать расход.
- `/income 4000 eur вторая работа` — записать доход.
- `/money` — сводка по доходам, расходам, долгам и целям.
- `/debt` — текущий план кредиток.
- `/reserve` — цели резерва.
- `/car` — авто-фонд и shortlist.
- `/canbuy 1800 щетка для бороды` — AI-оценка покупки.
- `/categories` — список категорий расходов.

## Быстрый старт

### 1. Установить зависимости

```bash
npm install
```

### 2. Поднять PostgreSQL

```bash
docker compose up -d
```

### 3. Создать `.env`

```bash
cp .env.example .env
```

Заполнить:

```env
TELEGRAM_BOT_TOKEN=
OPENAI_API_KEY=
DATABASE_URL=postgresql://opora:opora_password@localhost:5432/opora?schema=public
```

### 4. Сгенерировать Prisma Client

```bash
npm run prisma:generate
```

### 5. Создать миграцию и применить её

```bash
npx prisma migrate dev --name init
```

### 6. Запустить бота

```bash
npm run start:dev
```

## Создание Telegram-бота

1. Открыть Telegram.
2. Найти `@BotFather`.
3. Выполнить `/newbot`.
4. Задать имя и username.
5. Получить токен.
6. Вставить токен в `.env` как `TELEGRAM_BOT_TOKEN`.

## Финансовая логика MVP

Пока кредитки > 0:

1. Обязательные расходы.
2. Safe food сына.
3. Медицина / дети / семья.
4. Минимальный резерв.
5. Агрессивное закрытие кредиток.
6. Стиль / уход — умеренно.
7. Авто-фонд — пауза.

После кредиток = 0:

1. Резерв до 300 000 грн.
2. Резерв до 500 000 грн.
3. Авто-фонд.
4. Покупки уровня «хочу».
